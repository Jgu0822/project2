- `Java 11` `HTML5` `CSS3`
- `JDK 1.8.0`
- `IDE` : IntelliJ IDEA, Visual Studio Code
- `Framework` : Springboot(2.7.8)
- `Database` : MySQL
- `ORM` : JPA
