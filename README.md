* 23.03.14(화) - 23.04.06(목) (약 4주)

### 🧑‍🤝‍🧑 역할
 #### 팀원 : 장기운 
 - 결제관리 CRUD 구현
 - 조직관리 CRUD 구현
 - Komoran 기반 챗봇 기능구현 및 디자인

### ⚙️ 개발 환경
- `Java 11` `HTML5` `CSS3`
- `JDK 1.8.0`
- `IDE` : IntelliJ IDEA, Visual Studio Code
- `Framework` : Springboot(2.7.8)
- `Database` : MySQL
- `ORM` : JPA
